## Required Screenshots

Chrome Web Store requires at least 1 screenshot (1280x800 or 640x400).

### Recommended Screenshots

1. **popup-home.png** - Main popup interface showing iOS-style design
2. **instagram-before-after.png** - Split view showing feed with/without ads
3. **feed-settings.png** - Feed settings page
4. **blocked-counter.png** - Popup showing blocked items count
